# store-annual-report
Store Annual Sales Data Analysis
This project analyzes the annual sales performance of Vrinda Store using Microsoft Excel.
An interactive dashboard is created to help understand sales trends, customer behavior, order status, and channel performance for better business decision-making.
Objectives
Analyze monthly orders vs sales
Compare sales by gender
Understand order status distribution
Identify top-performing states
Analyze age group vs gender purchasing behavior
Evaluate sales contribution by sales channels
